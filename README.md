Sistema de Inventario Mi Pacífico V.WEB

Sistema de gestión de inventario desarrollado en Java Swing y MySQL para controlar productos, compras, ventas, movimientos de inventario, proveedores y usuarios dentro de una empresa.

Características
Gestión de productos.
Control de stock en tiempo real.
Registro de compras y ventas.
Historial de movimientos.
Administración de proveedores.
Gestión de usuarios y roles.
Reportes del sistema.
Alertas de productos con bajo inventario.
Tecnologías Utilizadas
Java 21
Java Swing
MySQL 8
JDBC
Maven
iText PDF
Módulos Principales
Productos

Permite registrar, actualizar, eliminar y consultar productos del inventario.

Compras

Registra compras a proveedores y actualiza automáticamente el stock.

Ventas

Gestiona ventas y descuenta existencias del inventario.

Movimientos

Mantiene un historial completo de entradas y salidas de productos.

Usuarios y Roles

Control de acceso mediante permisos según el rol asignado.

Reportes

Generación de reportes en PDF para análisis y control.

Requisitos
Java JDK 21 o superior
MySQL 8.0 o superior
NetBeans o IDE compatible
